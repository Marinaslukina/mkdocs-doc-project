    Пользовательский сценарий «Удаление заметки»

    Действующие лица:

    1. Пользователь

    2. Приложение

    3. Бэк

    4. База данных

    Предварительные условия

    Пользователь должен находиться на главном экране.

    Выходные условия

    Заметка удалена из системы.

    Основной сценарий

    Примечание.

    Сценарий добавлен с помощью переиспользования контента.

    1. Пользователь выбирает заметку, которую он хочет удалить, из списка заметок.

    2. Приложение открывает форму заметки.

    3. Пользователь нажимает кнопку «Удалить».

    4. Приложение открывает пользователю диалоговое окно «Удалить заметку?».

    5. Пользователь нажимает кнопку «Да» в диалоговом окне.

    6. Приложение отправляет запрос DELETE http://notesapp.su/api/notes/{id} Бэку на удаление заметки.

    7. Бэк удаляет заметку из Базы данных.

    8. Бэк возвращает Приложению ответ 204 «Заметка успешно удалена».

    9. Приложение открывает пользователю уведомление «Заметка успешно удалена!».

    Альтернативный сценарий

    1. Пользователь выбирает заметку, которую он хочет удалить, из списка заметок.

    2. Приложение открывает форму заметки.

    3. Пользователь нажимает кнопку «Удалить».

    4. Приложение открывает пользователю диалоговое окно «Удалить заметку?».

    5. Пользователь нажимает кнопку «Нет» в диалоговом окне.

    6. Приложение открывает форму заметки.

    7. Пользователь нажимает кнопку «Мои заметки».

    8. Приложение открывает окно со списком заметок.
