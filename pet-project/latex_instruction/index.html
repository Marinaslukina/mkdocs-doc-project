    Инструкция по работе с LaTeX

    Данная инструкция представляет собой пошаговую инструкцию по работе с LaTeX-шаблоном. Цель документа — стандартизация оформления текстовых материалов и максимальное упрощение процесса их создания для всех сотрудников, независимо от уровня владения LaTeX.

    В инструкции рассмотрены: базовая структура шаблона, порядок заполнения стандартных блоков (титульная страница, содержание, разделы).

    Для начала работы требуются только базовые навыки работы с текстовыми редакторами.

    Структура документа

    1. Для каждого нового проекта необходимо создать папку с наименованием проекта на английском языке, например, «OperatorManual».

    2. Далее в корневую папку вашего проекта необходимо скопировать предоставленный файл main.tex. Это главный файл, который управляет всем документом: в нём подключаются все остальные части (главы, стили оформления и прочее).

    3. В корневой папке следует создать подпапку img, в которой будут храниться все рисунки проекта.

    4. В корневой папке следует создать подпапку chapters. В данной подпапке будут лежать все главы документа, титульный лист, лист регистрации изменений, перечень принятых сокращений, введение.

    5. В подпапку chapters необходимо скопировать предоставленные файлы:

      • titel.tex – для формирования титульного листа документа;
      • abbreviations.tex – для создания перечня сокращений;
      • list.tex – для создания листа регистрации изменений.

    6. В корневой папке следует создать подпапку variables. В данную подпапку необходимо скопировать предоставленный файл var.tex. Данный файл содержит информацию для заполнения полей основного документа: децимальный номер изделия, код документа, вид документа, инвентарный номер, литера, год и т.д. Заполнение данного файла обязательно и будет указано ниже.

    7. Файл preample.tex содержит стилевые настройки для документа и не должен изменяться разработчиком. Данный файл необходимо скопировать в основную папку проекта.

    8. Файл list.tex содержит таблицу для создания листа регистрации изменений и не должен изменяться разработчиком.

    Подготовка файла с переменными данными

    1. Перед началом разработки необходимо заполнить файл с переменными данными – var.tex, находящийся в подпапке variables.

    2. Файл содержит следующий набор полей, значения которых следует указать:

      • наименование организации 

        \newcommand{\CompanyName}{ООО «Рога и копыта»}

      • наименование программы 

        \newcommand{\ProductName}{Программный комплекс}

      • децимальный номер программы 

        \newcommand{\ProductDocNumber}{АБВГ.12345-01}

      • код документа 

        \newcommand{\DocNumber}{34 01}

      • вид документа 

        \newcommand{\DocType}{Руководство оператора}

      • при наличии части документа указать часть (с маленькой буквы!), при отсутствии – поставить пробел 

        \newcommand{\DocPart}{часть 1}

      • год создания документа 

        \newcommand{\DocDate}{2025}

      • при наличии литеры указать ее, при отсутствии – поставить пробел 

        \newcommand{\DocLiter}{О}

      • при наличии инвентарного номера указать, при отсутствии – поставить пробел 

        \newcommand{\DocInventoryNumber}{11111}

    3. Данные, указанные в файле с переменными данными, автоматически будут заполнены на титульном листе и колонтитулах.

    4. При необходимости использовать значения переменных данных в тексте документа следует вместо значения указать его поле. Таким образом, при изменении значения в файле var.tex оно изменится во всем документе.

    Сборка проекта

    1. Сборка проекта осуществляется в файле main.tex.

    2. В данном файле осуществляется подключение всех вспомогательных файлов, которые составляют документ, в том порядке, в котором они должны быть расположены в документе.

    3. Файл уже содержит все основные команды для работы. Однако, список, наименование и порядок вспомогательных файлов может быть изменен разработчиком. Поэтому ниже приведено описание команд файла.

    4. В файле необходимо найти следующие команды, отмечающие начало и конец основного содержания документа: 

      \begin{document}  
<текст>
\end{document}

      Все, что находится между ними будет считаться частью основного текста документа.

      Далее, в этом разделе, речь пойдет о командах, которые следует записывать между началом и концом документа.

    5. Для подключения вспомогательного файла следует использовать следующую команду, где в фигурных скобках указывается путь до файла 

      \include{chapters/titel}

    6. Для создания новой страницы (разрыва страницы) необходимо использовать команду 

      \newpage

    7. Для создания содержания следует использовать ряд команд 

      \renewcommand{\contentsname}{}
\renewcommand{\cfttoctitlefont}{}
\mytableofcontents

    8. Для создания перечня сокращений следует использовать команду 

      \include{chapters/abbreviations}

    9. Для создания листа регистрации изменений следует использовать команду 

      \include{chapters/list}

    Основной текст документа

    1. Необходимо разделить содержимое документа на отдельные файлы, где каждая глава документа хранится в своем собственном файле. При большом объеме допускается делить на разделы.

    2. Наименование каждого файла должно состоять из английских букв, за которыми следует точка и расширение .tex. Например, «Introduction.tex», «Chapter1.tex» и т.д.

    3. Чтобы вставить наименование главы, необходимо использовать следующую команду 

      \chapter{Наименование главы}

      Пример:

      1 НАИМЕНОВАНИЕ ГЛАВЫ

    4. Чтобы вставить наименование раздела, необходимо использовать следующую команду 

      \section{Наименование раздела}

      Пример:

      2.1 Наименование раздела

    5. Чтобы вставить наименование пункта, необходимо использовать следующую команду 

      \subsection{Наименование пункта}

      Пример:

      2.1.1 Наименование пункта

    6. Чтобы вставить наименование подпункта, необходимо использовать следующую команду 

      \subsubsection{Наименование подпункта}

      Пример:

      3.1.4.2 Наименование подпункта

    7. Чтобы создать ненумерованный заголовок, включенный в содержание, необходимо:

      – для главы использовать команду 

      \notnumberchapter{Название главы}

      – для раздела использовать команду 

      \notnumbersection{Название раздела}

      – для пункта использовать команду 

      \notnumbersubsection{Название пункта}

      – для подпункта использовать команду 

      \notnumbersubsubsection{Название подпункта}

    8. Чтобы создать нумерованный заголовок, не включенный в содержание, необходимо:

      – для главы использовать команду 

      \notocchapter{Название главы}

      – для раздела использовать команду 

      \notocsection{Название раздела}

      – для пункта использовать команду 

      \notocsubsection{Название пункта}

      – для подпункта использовать команду 

      \notocsubsubsection{Название подпункта}

    9. Чтобы вставить рисунок в содержимое текста, необходимо:

      • сохранить рисунок в папку img;
      • в тексте указать следующий код 
        \begin{figure}[H]
\centering
\includegraphics[width=0.5\textwidth]{рисунок 14}
\caption{Пример отображения рисунка}
\label{fig:fig_14}
\end{figure}

      Поле \includegraphics в фигурных скобках {} должно содержать наименование рисунка без указания расширения.

      Поле \caption в фигурных скобках {} должно содержать подпись к изображению.

      Поле \label в фигурных скобках {} после значения fig: должно содержать произвольную метку, состоящую из английских букв и цифр. Данная метка будет использована для ссылки на изображение.

    10. Чтобы вставить ссылку на изображение в документе, необходимо использовать команду 

      \ref{fig:fig_1}

      Данная команда в фигурных скобках {} после значения fig: содержит метку из поля \label.

    11. Чтобы вставить ненумерованный список с маркером (–), необходимо в текст вставить следующий код: 

      \begin{itemize}
    \item текст перечисления 1;
    \item текст перечисления 2.
\end{itemize}

      Поле \item указывает на элемент списка.

    12. Чтобы вставить нумерованный список с маркером 1), необходимо в текст вставить следующий код: 

      \begin{enumerate}
    \item текст перечисления 1;
    \item текст перечисления 1.
\end{enumerate}

      Поле \item указывает на элемент списка.

    13. Для вставки неразрывного пробела необходимо использовать символ «~».

    14. Для вставки знака процента необходимо использовать сочетание символов «\%».

    15. Чтобы вставить длинное тире «–» необходимо использовать набор символов «--».

    16. Чтобы изменить начертание текста на жирное, необходимо использовать команду 

      \textbf{Текст жирным шрифтом}

    17. Для вставки сноски необходимо использовать команду 

      \footnote{Текст сноски}

    18. Для вставки иконки в текст следует:

      • сохранить иконку в папку img;
      • в тексте указать следующий код 
        \includegraphics[width=0.5cm]{Наименование иконки}

    19. Вставка таблицы, размещенной на одном листе

      • Чтобы создать таблицу и определить ее окружение, следует использовать команды:
      \begin{table}[H]
<текст>
\end{table}

      Указанные ниже команды должны быть записаны внутри окружения.

      • Для настройки и создания подписи к таблице следует вставить:
      \captionsetup{
    format=razryadka,
    justification=raggedright,
    singlelinecheck=false, 
    labelsep=endash,
}
\caption{Название таблицы 1}

      Поле \caption в фигурных скобках {} содержит наименование таблицы.

      • Для создания ссылки на таблицу необходимо добавить: 
      \label{tab:table_1}

      Поле \label в фигурных скобках {} после значения tab: должно содержать произвольную метку, состоящую из английских букв и цифр. Данная метка будет использована для ссылки на таблицу.

      • Для создания таблицы необходимо указать команды начала и конца таблицы:
      \begin{tabularx}
<текст>
\end{tabularx}
      • Для настройки ширины колонок следует добавить:
      \begin{tabularx}{\textwidth}{
    |>{\raggedright\arraybackslash}p{3.4cm}
    |>{\raggedright\arraybackslash}p{4.4cm}
    |>{\centering\arraybackslash}p{1.5cm}
    |>{\centering\arraybackslash}p{3cm}
    |>{\centering\arraybackslash}X|
}

      «|» – вертикальная линия (граница колонки);

      «>{}» – модификатор для установки стиля содержимого колонки;

      «\raggedright» – выравнивание текста по левому краю;

      «\centering» – выравнивание текста по центру;

      «\arraybackslash» – позволяет использовать знак «\» переноса;

      «p{3.4cm}» – задана фиксированная ширина колонки 3,4 см;

      «X» – автоматическая ширина колонки (растягивается на все доступное пространство).

      В примере выше определены пять колонок, четыре из которых имеют фиксированную ширину, а последняя заполняет оставшееся пространство до края листа.

      • Для создания горизонтальной линии следует использовать команду
      \hline

      • Для разделения столбцов между собой используется символ «&», для разделения строк – «\».

      • Для создания заголовка таблицы используется команда \makecell{}. Внутри фигурных скобок указывается название заголовка. Чтобы сделать переносы внутри слова, необходимо использовать символ «\».

      \makecell{Обозначение} & 
\makecell{Наименование} & 
\makecell{Коли-\\чество} & 
\makecell{Порядковый\\учетный\\номер} & 
\makecell{Примечание} \\ \hline\hline
      • Пример создания основного содержимого таблицы
      АБВГ.12345-01 20 01 & Ведомость эксплуатационных документов & 1 & 123 & * \\ \hline
      • Пример таблицы
      \begin{table}[H]
\captionsetup{
    format=razryadka,
    justification=raggedright,
    singlelinecheck=false, 
    labelsep=endash,
    }
\caption{Н\hspace{2pt}а\hspace{2pt}з\hspace{2pt}в\hspace{2pt}а\hspace{2pt}н\hspace{2pt}и\hspace{2pt}е \hspace{2pt}т\hspace{2pt}а\hspace{2pt}б\hspace{2pt}л\hspace{2pt}и\hspace{2pt}ц\hspace{2pt}ы\hspace{2pt} 1}
\label{tab:table_1}
\begin{tabularx}{\textwidth}{
    |>{\raggedright\arraybackslash}p{3.4cm}% вертикальная линия (граница колонки)
    |>{\raggedright\arraybackslash}p{4.4cm}%Х-колонка с автоматической шириной (растягивается)
    |>{\centering\arraybackslash}p{1.5cm}
    |>{\centering\arraybackslash}p{3cm}%автоматическая ширина, текст слева
    |>{\centering\arraybackslash}X|
}
\hline
\makecell{Обозначение} & \makecell{Наименование} & \makecell{Коли-\\чество} & \makecell{Порядковый\\учетный\\номер} & \makecell{Примечание} \\ \hline\hline
АБВГ.12345-01 20 01 & Ведомость эксплуатационных документов & 1 & 123 & * \\ \hline
АБВГ.12345-01 34 01 & Руководство оператора & 2 & 456 & ** \\ \hline
АБВГ.12345-01 96 01 & Комплект эксплуатационной документации & 3 & 789 & \textsuperscript{**} \\ \hline
\end{tabularx}
\end{table}
